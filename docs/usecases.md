# Описание сценариев

**Система:** Сайт wordpress.com.

**Акторы:** Гость, Пользователь.

## Сценарии:

### UC-01: Переход к странице начала работы.
- Система: Домашняя страница wordpress.com
- Действующее лицо: Гость
- Цель: Начало работы с WordPress (вход)
- Триггер: Нажатие на кнопку Начать сейчас
- Результат: Форма входа отображена на экране
- Связанный артефакт: [HomePageTest](../src/test/java/ru/itmo/tests/HomePageTest.java)

### UC-02: Авторизация пользователя.
- Система: Страница авторизации пользователя wordpress.com
- Действующее лицо: Зарегистрированный пользователь
- Цель: Вход в аккаунт
- Триггер: Ввод данных и нажатие кнопки Войти
- Результат: Доступ к личному кабинету, Пользователь на странице с сайтами
- Связанный артефакт: [AuthPageTest](../src/test/java/ru/itmo/tests/AdminPageTest.java)

### UC-03: Выход пользователя из аккаунта.
- Система: Страница личного кабинета пользователя wordpress.com
- Действующее лицо: Авторизованный пользователь
- Цель: Выход из аккаунта
- Триггер: Нажатие кнопки Выйти в выпадающем меню
- Результат: Пользователь не авторизован
- Связанный артефакт: [AuthPageTest](../src/test/java/ru/itmo/tests/AdminPageTest.java)

### UC-04: Создание сайта с ИИ.
- Система: Страница Сайты личного кабинета пользователя wordpress.com
- Действующее лицо: Авторизованный пользователь
- Цель: Создание сайта с ИИ
- Триггер: Нажатие на кнопку Создавайте с ИИ в меню Добавить новый сайт
- Результат: Доступ к чату с ИИ для создания сайта
- Связанный артефакт: [SitesPageTest](../src/test/java/ru/itmo/tests/SitesPageTest.java)

### UC-05: Удаление сайта.
- Система: Страница Сайты личного кабинета пользователя wordpress.com
- Действующее лицо: Авторизованный пользователь
- Цель: Удаление сайта
- Триггер: Нажатие на кнопку Удалить в меню управления сайтом
- Результат: Сайт удален, не показывается в списке сайтов пользователя
- Связанный артефакт: [SitesPageTest](../src/test/java/ru/itmo/tests/SitesPageTest.java)

### UC-06: Навигация по категориям личного кабинета.
- Система: Страница личного кабинета пользователя wordpress.com
- Действующее лицо: Авторизованный пользователь
- Цель: Перемещение между страницами личного кабинета
- Триггер: Нажатия на кнопки в меню навигации
- Результат: Отображение страниц личного кабинета
- Связанный артефакт: [AdminPageTest](../src/test/java/ru/itmo/tests/AdminPageTest.java)

### UC-07: Предпросмотр бесплатной темы.
- Система: Страница поиска тем wordpress.com
- Действующее лицо: Авторизованный пользователь
- Цель: Предпросмотр бесплатной темы из списка
- Триггер: Фильтрация тем, выбор одной и нажатие кнопки Предпросмотр на странице темы
- Результат: Тема отображена на экране
- Связанный артефакт: [ThemesPageTest](../src/test/java/ru/itmo/tests/ThemesPageTest.java)
